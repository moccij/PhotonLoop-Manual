Actuators Table

The actuators table widget presents the current state of the WFC actuators as a grid, with each row representing an actuator index.

Item Description

#

Actuator index.

Image

Actuator response wavefront.

Value

Current actuator command output.

Offset

Actuator command offset.

Unresolved directive in <stdin> - include::table.adoc[lines=5..-1]

The actuators widgets will become enabled once a WFC plugin is succesfully opened.

The actuator response is only shown after the AO system is calibrated.
