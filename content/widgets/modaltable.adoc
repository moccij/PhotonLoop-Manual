The *modal table* widget presents the current state of the modal reconstruction as a grid, with each row representing a mode.

[cols="1,3"]
|===
|#
|Mode index.

|Image
|Mode wavefront representation.

|Disabled
|Enable/disable the mode.

|Value
|Current measured mode.

|Offset
|Mode offset.
|===
