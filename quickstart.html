Quick Start

  • Installation

    • Package Contents The devices supplied by Dynamic Optics for a standard adaptive optical system should include: Item Description PhotonLoop Dongle key. This key will permit you to run PhotonLoop. USB Memory Stick Each product from Dynamic Optics is delivered with a USB key containing the principal information (manuals and datasheet, etc..) regarding the purchased products.

  • Wavefront Measurement

    • An accurate wavefront measurement is the starting point for a succesful adaptive optics correction. Follow this tutorial to grasp the basics of wavefront sensing within PhotonLoop. Sub-optimal measurement settings have a detrimental impact on the correction quality and might lead to an unstable system behavior.   • Aberration Correction

    • PhotonLoop connects to both wavefront sensor and corrector, providing the means of compensating measured aberrations. This tutorial will show how to correctly carry out the AO calibration of the AO system and succesfully close the control loop. Please read the safety instructions carefully before operating with the WFC. This tutorial assumes that PhotonLoop is already configured to optimally measure the wavefront. Refer to the wavefront measurement tutorial for how to setup the measurements.
